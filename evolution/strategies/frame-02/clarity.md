# Frame 2 strategy audit — Clarity

## Observed in the live Frame 1

- The generated mission said “smallest honest attempt,” but did not reliably name a single action, target, or artifact.
- Success language repeated the user’s proof text rather than giving a crisp verdict contract.
- Simplification prepended “Do only the first observable slice” without proving that scope decreased.
- Generated copy could produce “1 minutes.”

## Proposal

Compile from four inspectable templates. Each output starts with an action verb, names one real target or artifact, declares a numeric scope variable, supplies an explicit criterion, and ends at one through ten minutes. A simplification must lower that numeric scope.

## Acceptance signal

Compiler tests cover every template, singular/plural duration, deterministic output, and a strictly decreasing scope value.

## Consensus votes (1–5)

- Assisted experiment compiler: **5**
- Criterion-linked evidence and decision: **5**
- Recoverable local workspace: **4**
