# Learning audit — Frame 03

## Raw observation

Frame 2 linked successors, but it did not preserve a stable belief identity or summarize what repeated receipts collectively implied. The default smaller test could repeat the same low-information proof.

## Strategy

Preregister the belief, retain its ID through proof-pattern changes and successors, and choose the next proof for discrimination rather than size alone. Render a readable lineage from assumption through branches, action, observation, interpretation, confidence effect, decision, and successor. After two receipts, synthesize contradictions, repetition, and current support deterministically.

## Acceptance signal

- Belief ID never changes within a lineage.
- Duplicate proof configurations are flagged.
- Contradictions produce a mixed state, not a certainty claim.
- Conclude, replicate, and pivot are explicit after repeated evidence.
