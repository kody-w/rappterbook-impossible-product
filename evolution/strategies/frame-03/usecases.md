# Use-case audit — Frame 03

## Raw observation

Different “impossible” situations need different kits. A message probe, private preparation, accommodation request, constraint lookup, and dignified stop cannot be represented as the same outbound sprint.

## Strategy

Compile decision-grade kits with a usable handoff, then let the user select a barrier-aware route before freezing: act safely now, prepare privately, seek trusted support/accommodation, or stop. Preserve a concise local evidence brief so a returning user can understand the decision without rereading every field.

## Acceptance signal

- Ask/send produce copy-ready text only when safe.
- Make/check produce inspectable skeletons.
- Blocked successors can change action, route, or criterion.
- The decision brief is local and adds no sharing mutation.
